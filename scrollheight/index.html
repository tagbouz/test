横 base 320: zoom(1.5)後 480
= 縦 base 700: zoom後 x
x = 1050(scrollHeight)

これだとダメ
横 base 320: zoom(1.5)後 480
= 縦 base x: zoom後 700
x = 320 * 700 / 320 * zoom(1.5)
x = 700 / zoom
x = 466(scrollHeight)
要はzoom前のアスペクト比を保持するようにheightを計算してやれば良い
